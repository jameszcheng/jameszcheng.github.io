Projects

Constrained Text Generation Benchmark

Designed a 348-prompt benchmark (local, global, and hybrid constraints) with dual-path verification to quantify autoregressive constraint-following limitations across 6 LLMs on 7,889 expanded variants. Built SFT and DPO fine-tuning pipelines and meta-ablation analyses, confirming a steep difficulty gradient driven by planning bottlenecks.

Code
MuSR: Murder Mystery Reasoning Benchmark

Extended the MuSR benchmark (ICLR 2024 Spotlight) with dynamic belief update and counterfactual reasoning tracks, generating graph-structured mystery datasets to evaluate multi-step logical deduction in LLMs. Built streaming evaluation pipeline testing how models revise predictions across evidence rounds.

LLM-as-Planner vs RL on Procgen

Built hybrid RL-LLM agents with entropy-based mode switching, boosting task success rates up to 2x over RL-only baselines (PPO, DQN) across procedurally generated environments.
